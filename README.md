# Simple Banking Application

This project is a beginner friendly banking system built in Java.  
It allows a user to create an account, deposit money, withdraw money and check balance.

## Features
• Create a bank account  
• Deposit and withdraw  
• Check balance  
• Stores data in memory using ArrayList  
• Clean method based structure

## How to run
1. Open the project.
2. Navigate to src folder.
3. Run Main.java.

## Learning points
• Classes and objects  
• ArrayList usage  
• Encapsulation  
• Basic validation
