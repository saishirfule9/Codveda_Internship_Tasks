# Codveda Internship Tasks

This repository holds the work I completed during my internship at Codveda Technologies. It includes the tasks assigned to me, my approach for each one, and the final outcomes.

## Overview

The purpose of this repo is to show my progress during the internship and how I applied my skills in Java, Spring Boot, SQL, and related tools. Each task is placed in a separate folder with its own explanation, code, and supporting files.

## What this repository contains

* Source code for all assigned tasks
* Task descriptions and objectives
* My solutions with explanations
* Notes and learning points from each task

1. Clone the repo


4. Install dependencies using Maven
5. Run the application
