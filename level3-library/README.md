This is a console-based **Library Management System** developed using Java and JDBC.  
It allows managing books, users, and borrow/return transactions.

## Features

- Add, view, borrow, and return books  
- Manage users  
- Track borrow/return transactions  
- Connects to MySQL database via JDBC  

## How It Works

1. User selects an option from the menu  
2. Program performs the selected operation on the database  
3. Input is validated and appropriate messages are displayed  
4. Loop continues until user chooses to exit

- Object-Oriented Programming (Classes & Objects)  
- JDBC (Java Database Connectivity)  
- CRUD operations with SQL  
- Console-based input and output  
- Exception handling and database interaction
